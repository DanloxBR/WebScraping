# Web Scraping com Java

Projeto de **Web Scraping em Java** que coleta informações de ações do site Fundamentus utilizando a biblioteca Jsoup.

## 📌 Sobre o Projeto

Este projeto foi desenvolvido para praticar técnicas de **extração de dados de páginas web (web scraping)** utilizando Java.

A aplicação acessa uma página de detalhes de uma ação no site Fundamentus e extrai informações relevantes do HTML.

Exemplo de página utilizada:

https://www.fundamentus.com.br/detalhes.php?papel=PETR4

---

## 🚀 Tecnologias Utilizadas

- Java 8+
- Maven
- Jsoup
- HTML Parsing

Biblioteca principal:

- Jsoup — parser HTML para Java usado para extrair dados de páginas web.
